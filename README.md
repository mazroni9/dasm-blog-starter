# DASM Blog (Next.js)

- App Router
- Markdown posts in `content/`
- Arabic + English friendly (RTL supported automatically via CSS class)

## Scripts
- `pnpm dev` / `npm run dev`
- `npm run build`
- `npm start`

## Add a new post
Create a file in `content/` named `my-post.md` with frontmatter:

```
---
title: "عنوان المقال"
date: "2025-08-27"
lang: "ar"
---

نص المقال...
```
